# Три шурупа

Миниатюра про оценку сроков. Захотелось закрепить в виде текста.


{% note info "TLDR" %}

ожидал, что вкручу три шурупа за 15 минут, вышло — за 30. По ходу задачи всплыла куча неочевидных вещей, которые в сумме удвоили время.

{% endnote %}


## Задача

Повесить картину в спальне и повесить полочку в детской. Для картины нужно вкрутить один шуруп, для полочки — два.

- Просверлил третью дырку — 2 минуты
- Взял из коробки три дюпеля и три шурупа — 1 минута
- Вбил первый дюпель — 1 минута
- Вбил второй дюпель — 0,5 минут
- Вбил третий дюпель — 0,5 минут
- Закрутил первый шуруп — 0,5 минут
- Закрутил второй шуруп — 0,5 минут
